= Diagrams
:page-role: terminal

This page is here to validate that diagram rendering is enabled in the Antora pipeline.

NOTE: Antora uses Asciidoctor.js; instead of the Ruby `asciidoctor-diagram` gem, this site enables diagram blocks via `asciidoctor-kroki`.
